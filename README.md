# Socket.IO Demo with plain HTML and NodeJS
A web page demo for Socket.IO built with HTML, CSS, JS, NodeJS, for a school exam.
## Getting Started
### Clone the Repository
  ```bash
  git clone https://github.com/Mitchi-02/web-node-socketio.git
  ```
### Start the server
```bash
  npm install
  npm run dev
  ```
