# Chat SaaS

## Getting started

Don't forget to fill your env file and docker compose with env variables

#### Server
```
cd server
cp .env.example .env
npm / yarn / pnpm install
npm / yarn / pnpm dev
```
#### Demo App
```
cd demo-app
cp .env.local.example .env.local
npm / yarn / pnpm install
npm / yarn / pnpm dev
```
